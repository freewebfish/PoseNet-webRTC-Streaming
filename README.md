PoseNet-webRTC-Streaming
====================



Usage
-----

```
npm install
npm start
```

```
http://localhost:3000 접속

1. Broadcaster - click 'start' btn
2. Viewer - click 'start' btn
3. Viewer - click 'Start Exercise' btn
4. Do exercise! (only squat)
```
